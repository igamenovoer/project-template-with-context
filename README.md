# Project Template with Context

This is a template project that provides a structured context directory and OpenSpec workflow integration for AI-assisted development.

## Features

- Organized context directory structure for managing project knowledge
- OpenSpec workflow integration with skills and prompts
- Git submodule support for external dependencies
- Ready-to-use project scaffolding

## Usage

1. Use this template to create a new repository
2. Customize the context directories for your project needs
3. Update this README with your project-specific information
